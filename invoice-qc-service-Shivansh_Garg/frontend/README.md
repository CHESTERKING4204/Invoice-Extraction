# Invoice QC Frontend

React-based web console for invoice quality control.

## Setup

```bash
npm install
```

## Development

```bash
npm start
```

Opens at http://localhost:3000

## Build

```bash
npm run build
```

## Features

- Upload multiple PDF invoices
- Paste JSON for validation
- View validation results in table
- Filter by valid/invalid status
- Color-coded error badges
